# 🎮 Mini Game Project in C++

This project contains two classic console-based games developed in C++:

1. Tic Tac Toe
2. Snake Game

The project demonstrates the use of core C++ programming concepts such as arrays, loops, functions, conditional statements, and game logic.

---

## 📌 Features

### 🎯 Tic Tac Toe
- Two-player gameplay
- Dynamic board display
- Win detection
- Draw detection
- Replay option
- Input validation

### 🐍 Snake Game
- Real-time snake movement
- Food generation
- Score tracking
- Snake growth after eating food
- Collision detection
- Game Over screen
- Replay option

---

## 🛠 Technologies Used

- C++
- Object-Oriented Programming Concepts
- Console-Based Interface
